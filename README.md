## Project Overview
This repository contains the implementation and analysis of fundamental computer science algorithms applied to real-world healthcare scenarios. The project demonstrates how basic data structures and algorithms can be utilized to organize, search, and model physiotherapy data such as patient recovery timelines, pain rating scales, and exercise repetitions.

## Real-World Applications
The algorithms implemented in this notebook are tied directly to clinical use cases:
* **Fibonacci Sequence (Dynamic Programming):** Used to model safe, progressive overload in patient exercise plans.
* **Sorting Algorithms (Bubble, Selection, Insertion):** Utilized to triage patient data, such as organizing pain scale reports to identify high-risk patients.
* **Binary Search:** Applied to efficiently retrieve specific patient records from large, sorted clinical databases in logarithmic time.
